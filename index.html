Featured Project

Hope Drop – Blood Bank Management System

Managing blood donations and requests efficiently is critical, but often lacks a centralized digital solution. To address this, I built Hope Drop, a full-stack MERN application that allows users and hospitals to manage blood requests in a streamlined way.


Tech Stack

MongoDB, Express.js, React.js, Node.js


Integrations

NodeMailer (for email notifications)


Key Features

  • Role-based access for users and hospitals
  • Secure JWT authentication for login
  • Blood request and management system
  • Email notifications for user request approvals
  • Global state management using React Context API
  • Deployed on Render
